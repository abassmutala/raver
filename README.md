## :toolbox: Getting Started

### :running: Run Locally

Clone the project

```bash
https://github.com/abassmutala/raver
```
Go to the project directory
```bash
cd raver
```
Install dependencies
```bash
npm install
```
Start the server
```bash
npm run dev
```
